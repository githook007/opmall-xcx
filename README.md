# opmall-xcx

## 介绍说明：

使用小程序开发工具进行使用

只需要修改根目录下面的:siteinfi.js文件对应自己的小程序的域名



```
'siteroot': 'https://youhost/web/shoproot.php',   
'apiroot': 'https://youhost/web/shoproot.php?_mall_id=#',
    /*id=# "#"替换成你创建的商城ID*/
'domain': 'https://youhost'
```



修改小程序appid在根目录文件project.config.json/或者在小程序开发者工具中进行修改

```
"appid": "#####",
```
